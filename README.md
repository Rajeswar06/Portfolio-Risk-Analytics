# Portfolio Risk Analytics Engine
A Python-based Portfolio Risk Analytics Engine for Indian equity markets that provides portfolio performance analysis, risk metrics, CAPM analysis, Monte Carlo simulation, Value at Risk (VaR), stress testing, and automated PDF report generation through an interactive Streamlit dashboard.

Features
- Dynamic NSE stock analysis
- Dynamic benchmark selection (NIFTY 50, NIFTY BANK, NIFTY IT)
- Interactive Streamlit dashboard
- Monte Carlo price simulation
- Historical, Parametric, Conditional and Monte Carlo VaR
- CAPM Analysis
- Beta and Alpha calculation
- Sharpe Ratio
- Sortino Ratio
- Maximum Drawdown
- Stress Testing
- PDF report generation
- Interactive Plotly visualizations
